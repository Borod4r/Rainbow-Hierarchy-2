# Changelog

## [2.6.4] - 2026-07-22

### Added
- Added support for Unity 6000.3 and newer.
- Added Assembly Definitions (`Borodar.RainbowHierarchy` and `Borodar.RainbowHierarchy.Editor`) for runtime and editor scripts.
- Added `package.json` manifest for Unity Package Manager (UPM) support.

### Changed
- Refactored hierarchy item identification to use `EntityId` instead of `int`.
- Moved demo scenes out of the package directory to `Assets/Plugins/Borodar/RainbowHierarchyDemo`.

### Removed
- Dropped support for Unity versions older than 6000.3.

## [2.6.3] - 2026-07-22

### Added
- Initial open-source release.
